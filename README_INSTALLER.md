# 条目分类管理器 使用说明

## 欢迎使用条目分类管理器！

条目分类管理器是一个功能强大的桌面应用程序，帮助您高效管理条目和分类。

## 主要功能

- ✓ **条目管理**：添加、编辑、删除条目
- ✓ **分类管理**：创建分类，拖拽条目到分类
- ✓ **拖拽操作**：支持拖拽条目到不同分类
- ✓ **文件操作**：新建、打开、保存、另存为
- ✓ **撤销重做**：支持操作历史记录
- ✓ **搜索功能**：快速查找条目和分类
- ✓ **批量导入**：支持文本粘贴和文件导入

## 快捷键

- **Ctrl+N** - 新建文件
- **Ctrl+O** - 打开文件
- **Ctrl+S** - 保存文件
- **Ctrl+Shift+S** - 另存为
- **Ctrl+Z** - 撤销
- **Ctrl+Y** - 重做
- **Alt+F** - 文件菜单
- **Alt+E** - 编辑菜单
- **Alt+A** - 关于菜单

## 文件格式

应用使用 JSON 格式保存数据，您可以随时编辑和分享数据文件。
