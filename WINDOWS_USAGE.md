# Windows 版本使用说明

## 下载和安装

### 方式一：便携版（推荐）

1. **下载压缩包**
   - 文件名：`条目分类管理器-1.0.0-Windows-x64.zip`（完整版）
   - 文件名：`条目分类管理器-1.0.0-Windows-x64-Optimized.zip`（优化版）

2. **解压缩**
   - 右键点击压缩包
   - 选择"解压到当前文件夹"或"解压到..."

3. **运行应用**
   - 进入解压后的目录
   - 双击 `条目分类管理器.exe`
   - 首次运行时，Windows 可能会显示安全警告
   - 点击"更多信息" → "仍要运行"

### 方式二：安装版（需要 Windows 系统构建）

安装版需要使用 `scripts/build-installer.sh` 在 Windows 系统上构建。

详细构建说明请查看：[BUILD_INSTALLER.md](./BUILD_INSTALLER.md)

## 版本选择

### 完整版（206MB）
- **适用场景**：需要完整功能
- **包含内容**：
  - 55个语言包
  - FFmpeg 音视频支持
  - Vulkan 图形渲染
  - 完整调试文件

### 优化版（90MB）
- **适用场景**：日常使用，快速下载
- **包含内容**：
  - 仅中英文语言包
  - 核心功能完整
  - 无音视频功能
  - 无高级图形渲染

## 快捷方式创建

### 便携版创建桌面快捷方式

1. **方法一：右键发送到桌面**
   - 右键点击 `条目分类管理器.exe`
   - 选择"发送到" → "桌面快捷方式"

2. **方法二：手动创建**
   - 在桌面右键 → 新建 → 快捷方式
   - 浏览到 `条目分类管理器.exe`
   - 点击"下一步" → "完成"

3. **方法三：拖拽**
   - 按住 Ctrl 键
   - 将 `条目分类管理器.exe` 拖到桌面
   - 松开鼠标即可创建快捷方式

### 固定到任务栏

1. 右键点击桌面快捷方式
2. 选择"固定到任务栏"
3. 或者在应用运行时右键任务栏图标
4. 选择"将此程序固定到任务栏"

### 固定到开始菜单

1. 右键点击应用图标
2. 选择"固定到开始屏幕"

## 文件关联

应用已配置与 `.json` 文件关联，您可以：
- 双击 `.json` 文件直接在应用中打开
- 右键 `.json` 文件 → "打开方式" → "条目分类管理器"
- 设置为默认打开程序

## 卸载

### 便携版

直接删除解压后的文件夹即可：
1. 关闭应用
2. 删除整个文件夹
3. 删除桌面快捷方式（如有）

### 安装版

使用控制面板卸载：
1. 打开"设置" → "应用"
2. 找到"条目分类管理器"
3. 点击"卸载"
4. 按照提示完成卸载

## 常见问题

### Q: 首次运行时显示安全警告？
A: 这是正常的，因为应用未签名。点击"更多信息" → "仍要运行"即可。

### Q: 应用无法启动？
A: 
1. 检查是否为 64 位 Windows 系统
2. 尝试以管理员身份运行
3. 检查杀毒软件是否拦截
4. 重新下载并解压

### Q: 打开文件时乱码？
A: 确保文件编码为 UTF-8，应用使用 UTF-8 编码读写文件。

### Q: 数据文件保存在哪里？
A: 数据文件保存在您选择的位置，默认在应用目录下。

### Q: 如何备份数据？
A: 直接复制 `.json` 数据文件到安全位置即可。

### Q: 如何更新应用？
A: 下载新版本，解压覆盖旧版本即可（便携版）。

## 系统要求

- **处理器**：x64 架构
- **内存**：建议 4GB 或更高
- **磁盘空间**：至少 300MB 可用空间
